Консольное приложение FileRemover, позваляет удалять по указанному пути 
файлы и пустые директориикоторые давно редактировались.
Для выхода из программы введите любой символ.

Параметры для удаления задаются в файле config.xml

<?xml version="1.0" encoding="Windows-1251"?>
<configFile>
	<removeDirectories	period = "5000">
		<directory	path = ".\path_1"	mask = "mask1"		age = "100"/>
		...
		<directory	path = ".\path_n"	mask = "mask_n"	   	age = "100"/>
	</removeDirectories>
</configFile>

Параметры задаются атрибутами тега <directory>:
Атрибут period 	- задает частоту выполнения сканирования директорий
Атрибут path	- задает путь до директориив которой расположены файлы для удаления
Атрибут mask	- задает маску удаляемых файлов
Атрибут age	- задает возраст (в секундах!); файлы и директории старше age удаляются
 
Для первичной проверки можно создать bmp файл в папке test текущей директории

P.s. Для пути и маски МОЖНО использовать русские символы
